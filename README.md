## Projects

### 1. Sales Performance Analysis
**Tools:** SQL, Excel  

**Overview:**  
Analyzed retail sales data to evaluate regional performance, product profitability, discount impact, and year-over-year sales trends.  
The analysis emphasizes identifying revenue drivers and loss areas to support business strategy.

**Project Folder:**  
`sales-performance-analysis/`

**Key Highlights:**
- Profit margin analysis by product category
- Discount bucket analysis to identify loss-making discounts
- Year-over-year sales growth using SQL window functions
- Excel-based dashboards for reporting and analysis

---

### 2. HR Attrition Analysis
**Tools:** Power BI, DAX  

**Overview:**  
Analyzed employee attrition data to identify patterns and key drivers behind workforce turnover.  
The project separates high-level attrition metrics from deeper driver analysis to support HR decision-making.

**Project Folder:**  
`hr-attrition-analysis/`

**Key Highlights:**
- Overall attrition rate and workforce metrics
- Attrition comparison by department and gender
- Attrition risk analysis using count vs rate (combo chart)
- Driver analysis based on job role, overtime, income band, and tenure

## Repository Structure
Each project folder typically includes:
- `data/` → Raw dataset used for analysis
- `sql/` → SQL queries (where applicable)
- `excel/` or `powerbi/` → Reporting and dashboard files
- `charts/` → Final visual outputs
- `insights/` → Business-focused conclusions
